# Ciencia_de_Dados_Analytics_Seguros
Projeto de dados desenvolvido no Databricks, com foco em SQL, bibliotecas Python aplicadas à ciência de dados. Estrutura pipelines para processamento de grandes volumes de dados, envolvendo organização, modelagem e geração de métricas estratégicas, integrando ciência de dados e analytics no contexto de seguros para apoiar decisões.
